<p>开课吧《编程<a title="查看与 算法 相关的文章" target="_blank">算法</a>基础》课程是耿宇航老师讲授的《编程入门基础》的后续课程，在Java基础打扎实后，你将在这门课程中体会到更多编程的乐趣，体验信息技术的强大魅力：你将了解到Java中数据的实现方式，学会解决程序问题中经典的思考方式，掌握如何使用Java中标准对象实现常用的算法，并在不断的实战练手中磨练自己的编程技巧。</p>
